# Types

- What is a **type**?
    - A kind of data
    - **Types** have associated methods and operators
    - You can print or concatenate strings
    - You can do math on numbers

# Basic Math

Let's add some new types

- **int**
    - An integer
    - 0, 5, 71, -115, ...
- **float**
    - A decimal number
    - 5.2, 8.333, 9.6
    - Limited precision
